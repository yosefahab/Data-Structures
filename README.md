# Data Structures

## Basic data structures implementation in c++17.


## The implementation uses c++ Classes, Templates and utilizes a makefile to build the code.


### Type Definitions:

+ *mArrList*: **Array List**
+ *mBst*: **Binary Search Tree**
+ *mLStack*: **Stack**
+ *mLinkedList*: **Linked List**
+ *mQueue*: **Queue**
+ *mStack*: **Stack**
